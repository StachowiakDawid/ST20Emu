# STLite

The API defined in the `72-TDS-525-03` document corresponds to the **2.07 version of STLite/OS20**.

This is different that the `../ST20-document-suite` corresponding to the **2.10 version of OS20**.

We could assume that STLite/OS20 is a legacy name for the OS20.

Note: OS21 is a different, upgraded kernel.
